AI 도구가 개발자 생태계에 미치는 영향 분석

Stack Overflow와 GitHub AI 레포지토리를 중심으로

윤태웅
한양대학교 ERICA 스마트융합공학부 스마트ICT융합전공

Abstract

2022년 11월 ChatGPT 출시 이후 소프트웨어 개발 생태계가 급격히 변화했습니다. 본 연구는 AI 코드 생성 도구가 개발자 커뮤니티에 미친 영향을 실제 데이터를 통해 분석합니다. GitHub AI 레포지토리 185만 개와 Stack Overflow 질문 데이터를 기반으로, ChatGPT 출시 전후의 개발 패턴 변화를 정량적으로 측정하고 분석합니다.

분석 결과, Stack Overflow 질문 수는 ChatGPT 출시 후 51% 감소한 반면, GitHub AI 레포지토리는 141% 증가하여 강한 음의 상관관계(r = -0.933)를 보였습니다. 이는 개발자들이 전통적인 Q&A 플랫폼에서 AI 도구로 문제 해결 방식을 전환했음을 시사합니다.
